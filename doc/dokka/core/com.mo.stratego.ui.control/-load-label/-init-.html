LoadLabel(defaultText: String, deltaTime: Float, symbol: Char, showProgress: () -> Boolean, skin: Skin, maxCount: Int = 3)

Label that displays a load animation. The animation consists of the repetition of the defined symbol. Each frame adds another instance of the symbol and lasts for the defined deltaTime. One cycle is deltaTime * maxCount long.
